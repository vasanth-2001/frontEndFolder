    *form : it is used to get data from user

        get method : - form data will be visible in browser url.
                     - with get method we cannot sent confidential data.
                     - there is character limitation with get method to sent data through url.
                     - by default every method is a get method.

        post method :- form data wil be not vissible to user it will save in body object.
                     - with post method we can submit confidential data.
                     - there is no character limitation to submit the data.
